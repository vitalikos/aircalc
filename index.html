После получения API key от TON Builders — пошагово:

1. Откройте ваш index.html файла Mini App

Найдите секцию <head> или конец <body>.

2. Вставьте SDK именно СЮДА (в <head>):

xml
<!DOCTYPE html>
<html>
<head>
    <!-- ВАШ существующий код -->
    
    <!-- ДОБАВЬТЕ ЭТО: SDK Analytics -->
    <script async 
            src="https://tganalytics.xyz/index.js" 
            onload="initAnalytics()" 
            type="text/javascript">
    </script>
    
</head>
<body>
    <!-- Ваш код приложения -->
</body>

3. Добавьте функцию инициализации (перед закрывающим </body>):

xml
<script>
function initAnalytics() {
  window.telegramAnalytics.init({
    token: 'ВАШ_API_KEY_ОТ_TON_BUILDERS',     // ← ВСТАВЬТЕ ЗДЕСЬ
    appName: 'ВАШ_APP_NAME_ОТ_TON_BUILDERS'  // ← И ЗДЕСЬ
  });
}
</script>
</body>
</html>

4. Полный пример готового кода:

xml
<!DOCTYPE html>
<html>
<head>
    <script async src="https://tganalytics.xyz/index.js" onload="initAnalytics()" type="text/javascript"></script>
</head>
<body>
    <!-- Ваш React/Vue/HTML код здесь -->
    
    <script>
    function initAnalytics() {
      window.telegramAnalytics.init({
        token: 'abc123def456',        // ← ВАШ API KEY
        appName: 'aircalc'            // ← ВАШ APP NAME
      });
    }
    </script>
</body>
</html>

5. Дальнейшие действия:

  1. Сохраните файл

  2. Загрузите на Vercel/GitHub Pages (обновите деплой)

  3. Откройте Mini App в Telegram — совершите действия

  4. Проверьте аналитику в @DataChief_bot или TON Builders dashboard
